# 欧玛机械 (Ouma Machinery) 企业网站系统

这是一个基于 Node.js, Express, EJS 和 SQLite 构建的专业企业机械制造网站系统。包含前台展示和后台管理功能。

## 功能特性

- **多语言支持**:
  - 中文/英文双语一键切换
  - 全站内容国际化 (i18n)
- **前台展示**:
  - 首页 (Hero Banner, 核心优势, 推荐产品)
  - 产品中心 (产品列表, 分类展示)
  - 产品详情 (详细参数, 图片展示, **视频演示**)
  - 关于我们 (双语介绍)
  - 联系我们
- **后台管理**:
  - 管理员登录 (默认账号: `admin`, 密码: `admin123`)
  - 产品管理 (添加、编辑、删除产品)
  - 支持中英文双语数据录入
  - **多媒体上传**: 支持产品图片和演示视频上传

## 安装与运行

由于开发环境中未预装 Node.js，请按照以下步骤在您的本地环境中运行：

### 1. 安装 Node.js
如果您的电脑未安装 Node.js，请访问 [nodejs.org](https://nodejs.org/) 下载并安装最新 LTS 版本。

### 2. 安装依赖
打开终端（命令行），进入项目根目录，运行以下命令安装所需依赖：

```bash
npm install
```

### 3. 启动服务器
依赖安装完成后，运行以下命令启动网站：

```bash
npm start
```

### 4. 访问网站
打开浏览器，访问：
- 前台首页: `http://localhost:3000`
- 后台管理: `http://localhost:3000/admin/dashboard`

## 项目结构

- `server.js` - 程序入口文件
- `database.js` - 数据库配置 (SQLite)
- `models/` - 数据模型定义
- `routes/` - 路由处理 (前台和后台)
- `views/` - EJS 页面模板
- `public/` - 静态资源 (CSS, 图片, JS)
- `database/` - SQLite 数据库文件存储位置

## 注意事项

- 首次运行时，系统会自动创建一个 SQLite 数据库文件，并填充一些示例数据。
- 图片上传将存储在 `public/images/` 目录下。
